Adaptive Design VS Responsive Design

Q:

Apa itu adaptive dan responsive web design? Apakah kamu setuju bahwa teknik tersebut bagus? Mengapa atau mengapa tidak?

A:

Adaptive adalah design suatu web yang tidak responsive, design setiap media berbeda-beda, kalau kita flashback ke 5 tahun belakang, kita browsing dengan handphone maka url nya "m.domain.com"

Responsive adalah design suatu web yang all in di semua media, jaman sekarang banyak design suatu web yang menggunakan responsive.

Teknik adaptive maupun responsive sama sama bagus,tinggal tergantung kebutuhan. Menurut saya kalau suatu web ditujukan untuk banyak user dan database serta banyak trafic perharinya lebih baik menggunakan yang adaptive agar lebih optimize saat meload web itu sendiri serta performnya juga akan baik.

Sedangkan teknik responsive sebaliknya.

Q:

Apakah hadirnya smartphone dan tablet mengubah desain web secara signifikan? Bagaimana kira-kira teknologi baru akan mengubah desain web di masa depan?

A:

Tentu saja dengan hadirnya smartphone dan juga tablet merubah Design web secara Signifikan. karena ukuran media desktop, tablet dan juga smartphone masing - masing berbeda. Jadi harus menyesuaikan masing - masing ukuran media tersebut, agar tetap terlihat bagus di sisi UI/UX nya.

Mungkin di masa depan akan hadir suatu teknologi dimana pada saat mata kita terarah ke bawah maka akan otomatis terscrool sendiri tanpa harus menscrollnya menggunakan tangan atau pun mouse.

Q:

Bagaimana kira-kira orang yang memiliki kekurangan fisik menggunakan website? Apakah kita perlu menyesuaikan website untuk mereka juga? Solusi apa yang dapat mengatasi keadaan tersebut?

A:

Menurut saya Perlu, satu contoh apabila seseorang tidak mempunyai tangan, pastinya dia akan kesulitan untuk berseluncur di suatu web. Solusinya mungkin dengan teknologi command voice bisa membantu orang - orang yang tidak memiliki tangan tersebut, dan bisa jadi lahan bisnis juga.

Q:

Ide atau konsep apa yang sesuai denganmu saat mempelajari desain web?

A:

Menghadirkan User interface yang sederhana dan manis serta memperkaya untuk user experincenya juga.
